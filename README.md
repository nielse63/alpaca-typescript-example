## Installation

```bash
git clone https://github.com/nielse63/alpaca-typescript-example.git
cd alpaca-typescript-example
npm ci
```

## Usage

```bash
# compile source
npm run build

# run production script
npm start

# run scrip in dev
npm run dev
```
